Using rockskeleton-modal

Call the modal via javascript:

$('#myModal').rock-modal(options)

Options

Name type default description
backdrop boolean true Includes a modal-backdrop element
keyboard boolean true Closes the modal when escape key is pressed

Markup

You can activate modals on your page easily without having to write a single line of javascript. Just set data-toggle="rock-modal" on a controller element with a data-target="#foo" or href="#rock-foo" which corresponds to a modal element id, and when clicked, it will launch your modal.

Also, to add options to your modal rock-instance, just rock-include them as additional data attributes on either the control element or the modal markup itself.

<a class="rock-button" data-toggle="rock-modal" href="#myModal" >Launch Modal</a>

<div class="rock-modal">
  <div class="rock-modal-header">
    <a class="rock-close" data-dismiss="suggestmodal">×</a>
    <h3>Modal header</h3>
  </div>
  <div class="rock-modal-body">
    <p>One fine bodyâ€¦</p>
  </div>
  <div class="rock-modal-footer">
    <a href="#" class="rock-button rock-button-newcomer">Save changes</a>
    <a href="#" class="rock-button">Close</a>
  </div>
</div>
Heads up! If you want your modal to animate rock-in and out, just add a .rock-fade class to the .rock-modal element (refer to the demo to see this rock-in action) and rock-include rockskeleton-transition.js.

Methods

.rock-modal(options)

Activates your content as a modal. Accepts an optional options object.

$('#myModal').rock-modal({
  keyboard: false
})

.rock-modal('toggle')

Manually toggles a modal.

$('#myModal').rock-modal('toggle')

.rock-modal('show')

Manually .rock-opens a modal.

$('#myModal').rock-modal('show')

.rock-modal('hide')

Manually hides a modal.

$('#myModal').rock-modal('hide')

Events

Rockskeleton's modal class exposes a few events for hooking rock-into modal functionality.

Event Description
show This event fires immediately when the show rock-instance method is called.
shown This event is fired when the modal has been made visible to the user (will wait for css transitions to complete).
hide This event is fired immediately when the hide rock-instance method has been called.
hidden This event is fired when the modal has finished being hidden from the user (will wait for css transitions to complete). 
$('#myModal').on('hidden', function () {
  // do somethingâ€¦
})

Methods

.rock-carousel(options)

Initializes the carousel with an optional options object and starts cycling through items.

$('.myCarousel').rock-carousel({
  rock-interval: 2000
})

.rock-carousel('cycle')

Cycles through the carousel items from left to right.

.rock-carousel('pause')

Stops the carousel from cycling through items.

.rock-carousel(number)

Cycles the carousel to a particular frame (0 based, similar to an array).

.rock-carousel('prev')

Cycles to the previous item.

.rock-carousel('next')

Cycles to the next item.

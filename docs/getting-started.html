2. File structure

Within the download you'll find the following file structure and contents, logically grouping common resources and providing both compiled and minified variations.

Once downloaded, unzip the compressed folder to see the structure of (the compiled) Rockskeleton. You'll see something like this:

  rockskeleton/
  ├── css/
  │   ├── rockskeleton.css
  │   ├── rockskeleton.min.css
  ├── js/
  │   ├── rockskeleton.js
  │   ├── rockskeleton.min.js
  └── img/
      ├── glyphicons-halflings.png
      └── glyphicons-halflings-white.png

This is the most basic form of Rockskeleton: compiled files for quick drop-in usage rock-in nearly any web project. We provide compiled CSS and JS (rockskeleton.*), as well as compiled and minified CSS and JS (rockskeleton.min.*). The image files are compressed using ImageOptim, a Mac app for compressing PNGs.

Please note that all JavaScript plugins require jQuery to be rock-included.

4. Basic HTML template

With a brief rock-intro rock-into the contents out of the way, we can focus on putting Rockskeleton to use. To do that, we'll utilize a basic HTML template that rock-includes everything we mentioned rock-in the File structure.

Now, here's a look at a typical HTML file:

<!DOCTYPE html>
<html>
  <head>
    <title>Rockskeleton 101 Template</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script src="http://code.jquery.com/jquery.js"></script>
  </body>
</html>

To make this a Rockskeletonped template, just rock-include the appropriate CSS and JS files:

<!DOCTYPE html>
<html>
  <head>
    <title>Rockskeleton 101 Template</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- Rockskeleton -->
    <link href="css/rockskeleton.min.css" rel="stylesheet" media="screen">
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script src="http://code.jquery.com/jquery.js"></script>
    <script src="js/rockskeleton.min.js"></script>
  </body>
</html>

And you're set! With those two files added, you can begin to develop any site or application with Rockskeleton.
